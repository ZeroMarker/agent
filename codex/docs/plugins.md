# Plugins（插件）

插件是可安装的包，包含技能、命令、工具、MCP 配置、hooks、资源等。

## 插件结构

插件通过 `.codex-plugin/plugin.json` 清单定义。

## 内容

- Skills（技能）
- Commands（命令）
- Tools（工具）
- MCP 服务器配置
- Hooks
- Assets（资源）
- Marketplace 元数据

## 安装

通过 Codex CLI 安装和管理插件。

详见: https://developers.openai.com/codex/plugins/build#plugin-structure
