# Codex CLI 简介

**Codex CLI** 是 OpenAI 推出的轻量级编码代理，在你的计算机上本地运行。

## 产品形态

- **CLI** — 终端优先的本地仓库工作
- **IDE 扩展** — 编辑器集成（VS Code、Cursor、Windsurf）
- **Codex App** — 桌面应用（规划、审查、交互工作）
- **Codex Web** — 云端托管的并行/离线工作
- **Browser Use** — Codex 控制的 Web 测试
- **Chrome 扩展** — 使用用户的 Chrome 配置
- **Computer Use** — 控制桌面应用和 OS UI

## 核心能力

- 理解代码库结构和语义
- 规划并执行代码改动
- 运行 shell 命令和测试
- 在沙箱环境中安全执行
- 支持 MCP 协议集成外部工具
- 通过 AGENTS.md 自定义行为
