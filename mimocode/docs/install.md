# 安装与启动

## 前提条件

1. 一款现代终端模拟器：WezTerm、Alacritty、Ghostty、Kitty 等
2. 你想使用的 LLM 提供商的 API 密钥

## 安装

**Mac/Linux:**

```bash
curl -fsSL https://mimo.xiaomi.com/install | bash
```

**Windows:**

```bash
npm install -g @mimo-ai/cli
```

> Mac 用户强烈推荐使用 iTerm 或 VSCode Terminal。

安装完成后，[连接一个模型提供商](models-provider.md)，然后[运行你的第一个提示](first-prompt.md)。
