# 连接模型

MiMo Code 使用 [AI SDK](https://ai-sdk.dev/) 和 [Models.dev](https://models.dev) 支持 **75+ LLM 提供商**，并支持运行本地模型。

## 连接提供商

推荐使用 [MiMo Token Plan](https://platform.xiaomimimo.com/token-plan)：

1. 在 TUI 中运行 `/connect` 命令，选择 Xiaomi MiMo
2. 前往 [platform.xiaomimimo.com](https://platform.xiaomimimo.com/) 登录并添加账单信息，复制 API 密钥
3. 粘贴 API 密钥

## 选择模型

配置好提供商后，使用 `/models` 命令选择模型：

```
/models
```
