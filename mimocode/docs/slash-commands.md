# 斜杠命令速查

| 命令 | 快捷键 | 说明 |
|------|--------|------|
| `/connect` | — | 添加提供商 |
| `/compact` | `ctrl+x c` | 压缩会话（别名 /summarize） |
| `/details` | `ctrl+x d` | 切换工具详情 |
| `/editor` | `ctrl+x e` | 打开外部编辑器 |
| `/exit` | `ctrl+x q` | 退出（别名 /quit、/q） |
| `/export` | `ctrl+x x` | 导出为 Markdown |
| `/help` | `ctrl+x h` | 显示帮助 |
| `/init` | `ctrl+x i` | 创建/更新 AGENTS.md |
| `/models` | `ctrl+x m` | 列出可用模型 |
| `/new` | `ctrl+x n` | 新会话（别名 /clear） |
| `/redo` | `ctrl+x r` | 重做修改 |
| `/sessions` | `ctrl+x l` | 列出/切换会话 |
| `/share` | `ctrl+x s` | 分享会话 |
| `/themes` | `ctrl+x t` | 列出主题 |
| `/thinking` | — | 切换思考块可见性 |
| `/undo` | `ctrl+x u` | 撤销消息及文件更改 |
| `/unshare` | — | 取消分享 |
