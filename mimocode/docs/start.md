# 简介

**MiMo Code** 是一款面向开发者的 AI 编码代理。它能够理解你的代码库、规划改动、安全地编辑代码，并与你团队已有的工具协同工作。

## 从这里开始

- **[快速开始](install.md)** — 从安装到在终端中完成第一次实际修改
- **[连接模型](models-provider.md)** — 配置 API 密钥，接入你想使用的 LLM 提供商
- **更新日志** — https://github.com/XiaomiMiMo/MiMo-Code/releases

## 你可以用 MiMo Code 做什么

- **[理解你的代码](interaction.md)** — 理清仓库的整体结构，并找到合适的切入点
- **[规划并构建功能](modes.md)** — 明确改动范围，使用计划模式更有把握地推动大型任务上线
- **[审查更改](permissions.md)** — 检查 diff、运行检查，并在合并前发现问题
- **[自定义 MiMo Code](config-files.md)** — 使用符合你团队工作方式的规则、技能和命令
- **[使用不同的 Agents](agents.md)** — 配置专门的代理，自定义提示词、模型和工具访问权限
- **[排查与修复](troubleshooting.md)** — 复现问题、定位根本原因，并验证修复效果
