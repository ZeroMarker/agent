# 手机 Agent

这里整理移动端自动化、手机控制和 App 操作相关的 Agent 项目。

## 项目列表

| 名称 | 链接 | 备注 |
| --- | --- | --- |
| Open-AutoGLM | https://github.com/zai-org/Open-AutoGLM | 面向手机自动化任务的 Agent 项目。 |
| MobileAgent | https://github.com/x-plug/mobileagent | 面向移动设备 GUI 操作的 Agent 项目。 |

## 关注点

- 是否支持 Android 或 iOS。
- 是否需要真机、模拟器或 ADB。
- 是否依赖视觉模型识别界面。
- 能否稳定处理滑动、输入、弹窗和权限确认。
- 是否提供任务录制、回放或评测能力。
