# 工作流编排

这里记录 Agent workflow、低代码自动化和多步骤任务编排平台。

## 工具列表

| 名称 | 方向 | 备注 |
| --- | --- | --- |
| Dify | LLM 应用与工作流平台 | 适合构建聊天助手、知识库应用和可视化工作流。 |
| n8n | 自动化工作流平台 | 适合连接 API、Webhook、数据库和第三方服务。 |

## 适合场景

- 把多个 API 调用串成稳定流程。
- 为 Agent 增加人工审核、分支判断和重试逻辑。
- 搭建知识库问答、内容生成、消息通知等应用。
- 将本地脚本、数据库和 SaaS 服务接入自动化流程。
