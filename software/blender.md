# Blender

Blender 是开源 3D 创作软件，可用于建模、动画、渲染、材质、几何节点和脚本化生成。这里用于记录 Blender 与 AI Agent、自动化脚本、创作流程结合的资料。

## 可整理方向

- Python 脚本自动建模、布光和渲染。
- 使用 Agent 生成 Blender Python 脚本。
- 文生 3D、图生 3D 资产导入 Blender 的流程。
- 几何节点、材质节点和程序化场景生成。
- 批量渲染、资产整理和格式转换。

## 常用入口

- Python 控制台：适合快速执行小段脚本。
- Text Editor：适合保存和运行 Blender Python 脚本。
- Add-ons：适合把常用能力封装成插件。
- Geometry Nodes：适合程序化建模和可调参数资产。

## Agent 辅助任务

| 任务 | Agent 可做的事 | 人工需要确认 |
| --- | --- | --- |
| 生成场景 | 编写 Python 脚本创建物体、材质、灯光和相机 | 构图、比例、审美效果 |
| 修复脚本 | 根据报错修改 API 调用和对象引用 | Blender 版本兼容性 |
| 批量处理 | 自动导入、转换、命名、渲染文件 | 输出路径和资源授权 |
| 教程整理 | 将操作步骤转为脚本或清单 | 是否符合实际操作习惯 |
