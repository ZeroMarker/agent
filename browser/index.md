# 浏览器自动化

浏览器类 Agent 主要用于网页访问、表单填写、数据采集、自动测试和跨页面任务执行。

## 工具列表

| 名称 | 方向 | 备注 |
| --- | --- | --- |
| browser-use | 浏览器 Agent 框架 | 可用于让 Agent 操作浏览器完成网页任务。 |

## 常见场景

- 自动打开网页并完成点击、输入、提交等操作。
- 抓取页面结构化信息。
- 对本地 Web 应用做端到端检查。
- 辅助完成需要登录态或多步骤交互的网页任务。
